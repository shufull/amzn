# amzn

Overview

## Description

`amzn`はクリップボードにコピーされたAmazonの商品URLを短縮するPythonスクリプトです。

## Installation

```shell
$ wget https://raw.githubusercontent.com/shufull/amzn/master/amzn.py
$ pip install pyperclip
```
